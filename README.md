# Resume Projects
# Create the environment
```
python -m venv map
```
## Activate the environment
# On Windows
```
.\map\Scripts\activate
```
# On Linux or MacOS
```
source map/bin/activate
```
## Install the requirements file

```
pip install -r requirements.txt
```

## RUN COMMAND
```
python app.py
```

## TO DEACTIVATE VIRTUAL ENVIRONMENT
```
deactivate
```
